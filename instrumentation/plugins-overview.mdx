title: "Supported Plugins"
description: "Complete list of all dd-trace plugins organized by category."

dd-trace ships with plugins for 100+ popular Node.js libraries. Every plugin is enabled by default and activates automatically when you require the corresponding package. See [Automatic Instrumentation](/instrumentation/automatic) for configuration details.
